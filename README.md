# Clínica Prime — Sofia IA

Ecossistema de atendimento inteligente para clínicas de estética e odontologia, criado com Antigravity.

## Módulos

| Página | Descrição |
|--------|-----------|
| `landing.html` | Landing page comercial da Sofia IA |
| `index.html` | Chatbot Sofia com IA (Google Gemini) |
| `dashboard.html` | Painel de gestão com KPIs e gráficos |
| `crm.html` | CRM de leads qualificados |
| `GoogleAppsScript.js` | Backend para Google Sheets + Calendar |

## Stack

- Frontend estático (HTML, CSS, JavaScript)
- IA: Google Gemini API (configurada pelo usuário no onboarding)
- Integração: Google Apps Script → Sheets + Calendar

## Deploy

Hospedado na Vercel. A página inicial redireciona para `landing.html`.
